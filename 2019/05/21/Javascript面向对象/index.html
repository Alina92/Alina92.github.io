什么是面向对象编程

  • 用对象的思想写代码，就是面向对象编程
  • 一直使用的对象数组Array和时间Date

面向对象编程特点（OOP）

  • 抽象：抓住核心问题
  • 封装：只能通过对象来访问方法
  • 继承：从已有对象上继承出新的方法
  • 多态：多对象的不同形态

  • 方法（行为、操作）
